# 2.2.3. Настройки НДС

Для установки НДС на стороне киоска, нужно: \
1\. Перейти в настройки терминала в панели Nekassir\
2\. Добавить НДС в настройках типа оплаты \
3\. Перезагрузить киоск для применения настроек\
Настройки производятся после установки НДС в ККТ
