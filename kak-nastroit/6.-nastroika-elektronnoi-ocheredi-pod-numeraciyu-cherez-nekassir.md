# 6. Настройка электронной очереди под нумерацию через nekassir

Для того, чтобы на электронной очереди отображались номера заказов, которые nekassir показывает гостю при включенной нумерации со стороны сервиса в конфиг электронной очереди необходимо настроить объект&#x20;

"externalOrderDisplaySettings" : {\
"active" : true,\
"defaultDisplayOrderNumberLength": 4,\
"defaultLeadOrderNumberSymbol": "",\
"sources":\[]

}



Данный объект достаточно добавить в конец параметров конфига и перезагрузить очередь, тогда заказы, имеющие внешние номера, будут отображать именно их на электронной очереди
