# 6. Настройка электронной очереди под нумерацию через nekassir

Для того, чтобы на электронной очереди отображались номера заказов, которые nekassir показывает гостю при включенной нумерации со стороны сервиса(Внешний номер заказа), в конфиг электронной очереди необходимо настроить объект

"externalOrderDisplaySettings" : {\
"active" : true,\
"defaultDisplayOrderNumberLength": 4,\
"defaultLeadOrderNumberSymbol": "",\
"sources":\[]

}

Данный объект достаточно добавить в конец параметров конфига и перезагрузить очередь, тогда заказы, имеющие внешние номера, будут отображать именно их на электронной очереди\
\
Для настройки электронной очереди, работающей через **iiko web**, в конфиге нужно изменить параметр на "true", пример ниже

Конфиг находится по пути AppData\Roaming\iiko\CashServer\PluginConfigs\Resto.Front.Api.Arrivals.V8.1.0.54

"ExternalApiService": {\
"IsActive": true,\
"BaseUrl": ""\
},

Для более тонких настроек используйте мануалы iiko.
